## 1. Цель и постановка задачи

Разработать программный комплекс, который автоматизирует подготовку файлов для производства сувенирной и брендированной продукции БрГТУ (значки, бейджи, сувениры) с учётом фирменного стиля университета и возможностей лабораторного оборудования.

**Ключевые требования:**
- Поддержка персонализации (ФИО, группа/должность)
- Генерация файлов для разных типов продукции:
  - закатные значки Ø 25–58 мм
  - пластиковые/акриловые бейджи 85×55 мм
  - гравировка на дереве/металле
- Форматы: PNG (300 dpi), PDF (с точным размером страницы), в перспективе — SVG 
- Передача файлов на оборудование 

## 2. Текущая реализация 

**Технологический стек текущей версии:**
- Python 3.9+
- Pillow — генерация растровых изображений и PDF
- tkinter — простой графический интерфейс
- Модульная структура (набор компонент ПО)

## 3. Структура
Проект выполнен в модульной архитектуре — каждый файл отвечает за свою задачу:

- `main.py`              — точка входа и запуск программы  
- `ui.py`                — графический интерфейс (tkinter)  
- `badge_generator.py`   — логика генерации значков (бейджей)  
- `logo_generator.py`    — логика генерации вариантов логотипов  
- `utils.py`             — общие утилиты: шрифты, пути, константы  
- `assets/`              — папка с исходным логотипом университета (`logo.png`)  
- `output/`              — автоматически создаваемая папка с результатами
