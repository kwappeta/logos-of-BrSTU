## 1. Цель и постановка задачи

Разработать программный комплекс, который автоматизирует подготовку файлов для производства сувенирной и брендированной продукции БрГТУ (значки, бейджи, сувениры) с учётом фирменного стиля университета и возможностей лабораторного оборудования.

**Ключевые требования:**
- Персонализация: Генерация уникальных идентификаторов (ФИО, группа/должность).
- Мультиформатность: * 2D (PNG) — для закатных значков и печати бейджей.
- 3D (STL) — подготовка моделей с настраиваемой толщиной базы и рельефом логотипа.
- Инженерные параметры: Возможность ручной настройки физических размеров (в мм) для 3D-принтера.

## 2. Текущая реализация 

**Технологический стек текущей версии:**
- Python 3.10+.
- Pillow: Обработка растра и создание карт высот.
- NumPy: Математические вычисления для триангуляции поверхностей.
- numpy-stl: Формирование бинарных STL-файлов.
- Tkinter (ttk): Современный интерфейс на основе вкладок.
## 3. Структура
Проект выполнен в модульной архитектуре — каждый файл отвечает за свою задачу:
- main.py — инициализация приложения и запуск Event Loop.
- ui.py — описание интерфейса (класс App) и логика переключения вкладок.
- stl_generator.py — ядро системы: алгоритм превращения 2D-изображений в 3D-объекты.
-badge_generator.py — создание макетов персональных бейджей.
- logo_generator.py — генерация различных стилей логотипа БрГТУ.
- assets/ — база графических ресурсов университета.
-output/ — результирующие файлы (PNG, STL).
